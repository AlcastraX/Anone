HOW TO PLAY
GENERAL
  • Select Difficulties to Start
  • Select Modes in Settings
  • Use settings to customize your experience!
  • Press START to begin.
  • The number will be visible for just a few seconds before it become "?"
  • Answer the math question before time runs out.
  • Answer the questions correctly!
  • Keep your streak going for higher scores!
DIFFICULTIES
  • CASUAL is for beginners. It has a smaller number range and less Number Visibility Time (NVT)
  • INTERMEDIATE is for those who want a bit more challenge. It has a decent number range and more NVT
  • CHALLENGE is for those who want a real challenge. It has an even bigger number range and even more NVT
  • MASTERY...Your limit is tested by the System, your capability is determined by how much streak you earns. The Question difficulty, NVT, and time limit is determined by your streak.
SETTINGS
  • Allow Jitter adds jitter animation to buttons when hovered or clicked.
  • Allow Overlay adds a gray overlay on the screen when the settings menu is open, making it easier to focus on the settings.
  • Allow Negative allows negative answers in subtraction problems. If disabled, all subtraction problems will be arranged to have non-negative answers. Example(3-4 =|3-4| = 1)
  • Allow Decimal allows decimal answers in division problems. To answer decimal problems, you have to write 2 numbers behind the dot, example(4/3 = 1.33). If disabled, all division problems will be arranged to have integer answers.
  • Select Mode allows you to select the type of math problems you want to solve, including addition, subtraction, multiplication, and division.
  • This Game can have more features in the future
  • If you still want to read The rules and How to play, refresh is the way!
